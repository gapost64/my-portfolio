# Οδηγός για Αρχάριους

Καλώς ήρθατε στο έργο μου! Αυτός ο οδηγός θα σας βοηθήσει να ξεκινήσετε με το Git και να διαχειριστείτε αυτό το έργο. Ακολουθήστε τα παρακάτω βήματα:

---

## Πώς να ξεκινήσετε

1. **Εγκατάσταση του Git**:
   - Κατεβάστε και εγκαταστήστε το Git από την επίσημη ιστοσελίδα: [https://git-scm.com/](https://git-scm.com/).

2. **Αρχικοποίηση του Git**:
   - Αν δεν έχετε ήδη αρχικοποιήσει το repository, εκτελέστε την εντολή:
     ```bash
     git init
     ```

3. **Κλωνοποίηση του έργου**:
   - Ανοίξτε το τερματικό σας και εκτελέστε την εντολή:
     ```bash
     git clone <URL_του_αποθετηρίου>
     ```

4. **Μετακίνηση στον φάκελο του έργου**:
   - Μεταβείτε στον φάκελο του έργου με την εντολή:
     ```bash
     cd my-portfolio
     ```

---

## Βασικές Εντολές Git

1. **Έλεγχος κατάστασης**:
   - Δείτε τις αλλαγές στο έργο σας:
     ```bash
     git status
     ```

2. **Προσθήκη αλλαγών**:
   - Προσθέστε αρχεία για commit:
     ```bash
     git add .
     ```

3. **Δημιουργία commit**:
   - Αποθηκεύστε τις αλλαγές σας:
     ```bash
     git commit -m "Περιγραφή των αλλαγών"
     ```

4. **Αποστολή αλλαγών**:
   - Στείλτε τις αλλαγές στο αποθετήριο:
     ```bash
     git push
     ```

5. **Λήψη αλλαγών**:
   - Ενημερώστε το τοπικό σας αποθετήριο:
     ```bash
     git pull
     ```

---

## Συντήρηση του Έργου

1. **Ενημέρωση εξαρτήσεων**:
   - Ελέγξτε αν υπάρχουν ενημερώσεις για τις εξαρτήσεις του έργου.

2. **Δοκιμή αλλαγών**:
   - Πριν στείλετε αλλαγές, βεβαιωθείτε ότι το έργο λειτουργεί σωστά.

3. **Καθαρισμός κώδικα**:
   - Διατηρήστε τον κώδικα καθαρό και οργανωμένο.

---

## Εκκίνηση Live Server

1. **Εγκατάσταση του Live Server Extension**:
   - Εγκαταστήστε την επέκταση "Live Server" από το Visual Studio Code Marketplace.

2. **Εκκίνηση του Live Server**:
   - Ανοίξτε το αρχείο `index.html` στο Visual Studio Code.
   - Κάντε δεξί κλικ στο αρχείο και επιλέξτε "Open with Live Server".

3. **Προβολή στο πρόγραμμα περιήγησης**:
   - Το έργο σας θα ανοίξει αυτόματα στο προεπιλεγμένο πρόγραμμα περιήγησης στη διεύθυνση:
     ```
     http://127.0.0.1:5500
     ```

---

Ελπίζω αυτός ο οδηγός να σας βοηθήσει να ξεκινήσετε και να διαχειριστείτε το έργο σας! Αν έχετε απορίες, μη διστάσετε να ρωτήσετε.
